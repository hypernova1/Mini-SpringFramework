# Mini SpringFramework
jsp 프로젝트로 간단한 스프링 프로젝트를 구현한다.

## 목적
* 스프링 내부 구조 파악
* 객체 지향 학습
* Reflection 학습

## 구현(*리팩토링, 보완해야할..*)된 기능들
* 기본 기능
    * Converter
    * ViewResolver
    * DispatcherServlet
    * RequestHandlerMapping
    * RequestHandlerAdaptor
    * Model
    * BeanContainer

* Annotation
    * @Controller
    * @RequestMapping (@GetMapping, @PostMapping 등..)
    * @RequestBody, @ResponseBody
